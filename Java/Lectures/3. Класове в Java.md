# Класове в Java - Наследяване. Създаване и използване на обекти.

## 1. Класове (Classes)
Класът е шаблон за създаване на съответни обекти – описва полета (състояние) и методи (поведение).

---

### 1.1 Полета и Капсулация (Encapsulation)
- Полетата описват състоянието на обекта и обикновено са `private`.
- Достъпът е чрез **getter / setter** методи от `private` до `public`

    ```java
    public String getName() { 
        return this.name; 
    }
    public void setName(String name) { 
        this.name = name; 
    }
    ```


---

### 1.2 Конструктори
- Използват се за инициализиране на обекта.
- Нямат тип на връщане, **имат същото име като класа**.

**Видове:** 
- **Конструктор по подразбиране** (без параметри, създава се автоматично, ако няма друг)
- **Конструктор с параметри**

```java
public Cat(String name, int age) {
    this.name = name;
    this .age = age;
}
```

---

### 1.3 Ключова дума `this`
- Разграничава полета от параметри.
- Може да извика друг конструктор в съ щия клас.

```java
this(name, 0);
```

---

## 2. Създаване и използване на обекти
- Обектите се създават с `new`.
- Съхраняват се в **Heap**; променливата пази референция в Stack.

    ```java
    Cat myCat = new Cat("Whiskers", 5);
    ```

---

## 3. Наследяване (Inheritance)
- Създава връзка “is-a” и позволява повторно използване на код.
- Декларира се чрез `extends` (само ед ин родител е позволен).

    ```java
    class SubClass extends SuperClass {
        // SubClass наследява членовете на SuperClass
    }
    ```
**Ограничение:** Java поддържа **единично наследяване** (един клас може да наследи само един друг клас).

---

### 3.1 Ключова дума `super`
- Използва се в дъщерния клас.
- Дава достъп до членове на родителя.
- Първи ред в конструктора, ако се извиква конструктор на родителя.

    ```java
    public Dog(String name) {
        super(name); // Извиква конструктора на родителя
    }
    ```

--- 

### 3.2 Презаписване на Методи (Method Overriding)
- Дъщерният клас променя наследен метод със същата сигнатура.
- Препоръчва се анотацията `@Override`.

```java
@Override
public void makeSound() { System.out.println("Woof!"); }
```

---

### 3.3 `final`
- `final` поле → константа.
- `final` метод → не може да бъде override-нат.
- `final` клас → не може да бъде наследяван.

```java
final class Constants { }
```
