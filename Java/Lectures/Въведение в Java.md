# Въведение в Java – Примитивни типове и променливи, оператори, вход/изход, цикли и масиви

## Примитивни типове и променливи

**Променливите** съхраняват временни данни. В Java има два вида типове:
- **Примитивни** (в стека): `byte, short, int, long, float, double, boolean, char`
- **Референтни** (в heap): `Object, String`, масиви, класове и др.

**Бележки:**
- Ползва се **най-малкия възможен тип** когато е уместно.
- Внимава се за **загуба на точност** при `float/double`.
- `String` е **референтен тип**.
- Примитивите имат стойности по подразбиране (напр. `int -> 0`, `boolean -> false`, `String -> null`).

---

### Пример:

```java
byte x = 68;
int i = 123456;
char ch = 'W';
boolean result = false;
Object obj = 172;
String str = "Java is easy to learn";
```

При референтния тип в стека се съхранява **адрес**, който сочи към обекта в heap-а. Този адрес се нарича **референция** (аналогично на указателите в C).

![Logo](http://javac.bg/wp-content/uploads/2015/09/picture1.jpg)

Схемата показва как Java съхранява **примитивни** и **референтни** типове данни в паметта.
- **Примитивните типове** (`byte, int, char, boolean`) се съхраняват директно в **стека (Stack)**.
- **Референтните типове** (като `Object`, `String`, масиви и др.) се съхраняват в **heap паметта**, а в стека стои само **адресът (референцията)** към тях.
- Променливите `obj` и `str` в стека не съдържат стойността, а **адрес към обектите в heap-а**.
- Така Java разделя съхранението:  
  - *Stack → стойност или адрес*  
  - *Heap → реалният обект*

## Стандартни обхвати на примитивните типове в Java

| Primitive Type | Size     | Minimum Value                              | Maximum Value                              | Wrapper Type |
|----------------|----------|--------------------------------------------|--------------------------------------------|--------------|
| `char`         | 16-bit   | Unicode 0                                  | Unicode 2¹⁶ − 1                            | `Character`  |
| `byte`         | 8-bit    | -128                                       | +127                                       | `Byte`       |
| `short`        | 16-bit   | -2¹⁵ (-32,768)                             | +2¹⁵ − 1 (32,767)                          | `Short`      |
| `int`          | 32-bit   | -2³¹ (-2,147,483,648)                      | +2³¹ − 1 (2,147,483,647)                   | `Integer`    |
| `long`         | 64-bit   | -2⁶³ (-9,223,372,036,854,775,808)          | +2⁶³ − 1 (9,223,372,036,854,775,807)       | `Long`       |
| `float`        | 32-bit   | 32-bit IEEE 754 floating-point numbers     | 32-bit IEEE 754 floating-point numbers     | `Float`      |
| `double`       | 64-bit   | 64-bit IEEE 754 floating-point numbers     | 64-bit IEEE 754 floating-point numbers     | `Double`     |
| `boolean`      | 1-bit    | `true` or `false`                          | `true` or `false`                          | `Boolean`    |
| `void`         | —        | —                                          | —                                          | `Void`       |

### Важни моменти

- Използва се тип, заемащ по-малко памет, когато е възможно.  
- Внимава се при числа с плаваща запетая (възможна загуба на точност).  
- `char` представлява 16-битов Unicode символ.

## Оператори и изрази

### Унарни оператори в Java
| Оператор | Име / Действие              | Кратко обяснение |
|----------|-----------------------------|------------------|
| `++`     | Инкремент                   | Увеличава стойността с 1 |
| `--`     | Декремент                   | Намалява стойността с 1 |
| `+`      | Унарно плюс                 | Прави числото положително (без промяна) |
| `-`      | Унарно минус                | Променя знака (положително ↔ отрицателно) |
| `~`      | Побитово NOT (инвертиране)  | Обръща всички битове (0 → 1, 1 → 0) |
| `!`      | Логическо отрицание         | Обръща `true` → `false` и обратно |
| `(тип)`  | Преобразуване на тип (cast) | Принудително превръщане на стойност в друг тип |

### Аритметични оператори
`+`, `-`, `*`, `/`, `%`

### Оператори за сравнение
`<`, `<=`, `>`, `>=`, `==`, `!=`

### Логически оператори
`&&` – логическо "и"  
`||` – логическо "или"

### Побитови оператори в Java
| Оператор | Име / Действие | Кратко обяснение | Пример |
|----------|----------------|------------------|---------|
| `&`      | Побитово AND   | Резултатът е 1 само ако **и двата бита са 1** | `6 & 3` → `110 & 011 = 010 (2)` |
| `\|`     | Побитово OR    | Резултатът е 1 ако **поне един бит е 1** | `6 \| 3` → `110 \| 011 = 111 (7)` |
| `^`      | Побитово XOR (изключващо или) | Резултатът е 1 ако **битовете са различни** | `6 ^ 3` → `110 ^ 011 = 101 (5)` |
| `<<`     | Изместване наляво | Мести всички битове наляво → умножение по 2ⁿ | `6 << 1` → `1100 (12)` |
| `>>`     | Изместване надясно (със знак) | Запазва знака (попълва със знаков бит) | `-8 >> 1` → остава отрицателно |
| `>>>`    | Изместване надясно (без знак) | Принуждава попълване с 0 (винаги положително) | `-8 >>> 1` → става голямо положително число |

## Цикли в Java

Циклите се използват за повторение на съвкупност от операции.

### For цикъл
```java
for(int i = 0; i < 10; i++) {
    System.out.println("i is: " + i);
}
```

### While и Do-While
```java
int counter = 0;
while(counter < 10) {
    System.out.println("counter: " + counter);
    counter++;
}
```

### For-Each цикъл
```java
String[] strings = {"one", "two", "three"};
for(String s : strings) {
    System.out.println(s);
}
```

## Обхождане на масив

```java
String[] stringArray = new String[10];
for(int i = 0; i < stringArray.length; i++) {
    stringArray[i] = "String no " + i;
```

## Стандартен вход и изход

### Изход (конзола)
```java
System.out.print("Some text.");
System.out.println("Some text.");
System.out.println("text 1 " + "text 2");
System.out.println("x = " + x);
System.out.format("%s", "Hello World\n");
```

---

### Вход (чете от клавиатура)

#### Пример с `System.in.read()`:
```java
import java.io.IOException;

public class MyFirstProgram {
    public static void main(String[] args) throws IOException {
        int input;
        do {
            input = System.in.read();
            System.out.print((char) input);
        } while (input != '\n');
    }
}
```

#### Пример с `Scanner`:
```java
import java.util.Scanner;

Scanner scanner = new Scanner(System.in);
String name = scanner.nextLine();
int age = scanner.nextInt();
double value = scanner.nextDouble();
```

[Документация за Scanner](https://docs.oracle.com/javase/7/docs/api/java/util/Scanner.html)

## Структура на Java програма

Обикновеният Java проект представлява директория, съдържаща `.java` файлове (сорс код).  
След компилация се генерират `.class` файлове – байт код, който се изпълнява от **Java Virtual Machine (JVM)**.

**IDE примери:** Eclipse, NetBeans, IntelliJ IDEA, DrJava.

### Основен метод
```java
public static void main(String[] args) {
    // entry point
}
```
