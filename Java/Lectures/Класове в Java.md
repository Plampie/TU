# Класове в Java - Наследяване. Създаване и използване на обекти.

## 1. Класове (Classes)

Класът е основната градивна единица в Java и е абстрактно описание на даден обект.

* **Определение:** Класът е **шаблон, проект или чертеж** за създаване на обекти. Той дефинира общите характеристики (**полета**) и поведение (**методи**) за всички обекти, които ще бъдат създадени от него.
* **Членове на Класа:** Полетата, конструкторите и методите са общо известни като **членове** (*members*) на класа.

---

### 1.1. Деклариране на Полета и Капсулация

Полетата представляват **състоянието** на обекта.

* **Капсулация (Encapsulation):** Добра практика е полетата да се декларират като **`private`**, за да се скрие вътрешната им структура от външния свят.
* **Гетери и Сетери (Getters and Setters):** Достъпът до частните полета става чрез **публични методи** (гетери за четене, сетери за промяна), които контролират този достъп.
    ```java
    public String getName() { 
        return this.name; 
    }
    public void setName(String name) { 
        this.name = name; 
    }
    ```

---

### 1.2. Конструктори (Constructors)

Конструкторът е **специален метод** за **инициализация** на полетата при създаването на обект.

* **Име:** Има **същото име** като класа и **няма** тип на връщана стойност.
* **Видове:** Може да бъде **Конструктор по подразбиране** (без параметри, създава се автоматично, ако няма друг) или **Конструктор с параметри**.

---

### 1.3. Ключовата дума `this`

Ключовата дума **`this`** се използва в класа за:

1.  **Разграничаване:** Разграничава **полето на класа** от локален параметър/променлива с едно и също име (напр. `this.name = name;`).
2.  **Извикване на Конструктор:** Извиква друг конструктор в същия клас (напр. `this(name, 0);`). Трябва да е първият ред.

---

## 2. Създаване и Използване на Обекти

Обектът е конкретна **инстанция** (*instance*) на класа.

* **Процес:** Извършва се чрез ключовата дума **`new`** и извикване на Конструктора.
    ```java
    Cat myCat = new Cat("Whiskers", 5);
    ```
* **Памет:** Обектът се създава в паметта **Heap** (купчина), а променливата `myCat` съхранява **референция** (адрес) към него в паметта Stack. 
* **Достъп:** Достъпът до членовете на обекта (полета и методи) става чрез **оператора точка** (`.`).

---

## 3. Наследяване (Inheritance)

Наследяването позволява един клас (**дъщерен**) да придобие характеристиките и поведението на друг клас (**родителски**).

* **Определение:** Създава връзка **"is-a"** (е-вид-на) и осигурява **Повторно използване на код** (*Code Reusability*).
* **Синтаксис:** Използва се ключовата дума **`extends`**.
    ```java
    class SubClass extends SuperClass {
        // SubClass наследява членовете на SuperClass
    }
    ```
* **Ограничение:** Java поддържа **единично наследяване** (един клас може да наследи само един друг клас).

---

### 3.1. Ключовата дума `super`

**`super`** се използва в дъщерния клас за достъп до членовете на родителския клас.

* **Конструктор:** Извиква конструктора на родителския клас. **Трябва** да бъде първият ред в конструктора на дъщерния клас.
    ```java
    public Dog(String name) {
        super(name); // Извиква конструктора на родителя
    }
    ```

---

### 3.2. Презаписване на Методи (Method Overriding)

* **Определение:** Дъщерният клас дава **своя собствена имплементация** на метод, наследен от родителския клас.
* **Изискване:** Методът в дъщерния клас трябва да има **абсолютно същата сигнатура** като метода в родителския клас.
* **Препоръка:** Използва се анотацията **`@Override`**.

---

### 3.3. Ключова дума `final`

`final` се използва за **забрана** на промяна или наследяване:

* **`final` поле:** Декларира константа, чиято стойност не може да се променя.
* **`final` метод:** Не може да бъде **презаписан** (*overridden*).
* **`final` клас:** Не може да бъде **наследен**.
