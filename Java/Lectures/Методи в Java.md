# Въведение в Java – Примитивни типове и променливи, оператори, вход/изход, цикли и масиви

## 1. Какво е метод?

**Метод** наричаме обособена част от програмата, която изпълнява конкретна задача.  
Методите:
- повишават четимостта на кода
- позволяват повторна употреба (code reuse)
- разделят логиката на по-малки части („разделяй и владей“)
- са основен елемент в ООП – описват поведението на обектите
